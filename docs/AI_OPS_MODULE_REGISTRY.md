## 9. Purpose of the Registry

- A reference for AI agents loading modules  
- A structured overview of the entire framework  
- A consistency layer ensuring naming conventions remain aligned  

---

## 10. Updating the Registry

Whenever a new module is added:

1. Add it to the correct section  
2. Maintain alphabetical order  
3. Use the correct naming convention (`ai-ops-[type]-[name].md`)  
4. Keep descriptions concise  
5. Ensure the file is also added to `ai-ops-manifest.json`  
6. Confirm links and paths resolve correctly
