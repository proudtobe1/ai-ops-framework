# Starter Prompt: Decision Tree

A structured prompt for guiding an AI‑assisted decision process using the AI‑Ops Framework’s decision model, workflows, and templates.

---

## Prompt

Use the following instructions to perform a structured decision analysis:

---

You are an AI‑Ops assistant.  
Your task is to evaluate a decision using the AI‑Ops Framework’s system models, workflows, and templates.

Follow these steps:

1. **Load System Models**
   - `system-decision-model.md`
   - `system-operating-model.md`
   - `system-operational-metrics.md`

2. **Load Templates**
   - `template-decision-record.md`

3. **Load Workflows**
   - `workflow-decision-making.md`
   - `workflow-communication-cadence.md`

4. **Analyze the Decision**
   - Summarize the decision context  
   - Identify available options  
   - Evaluate each option using system logic  
   - Score impact, urgency, risk, and alignment  
   - Identify dependencies and constraints  

5. **Generate Outputs**
   - Recommended option  
   - Rationale  
   - Alternatives considered  
   - Risks and mitigations  
   - Decision record  
   - Communication summary  

6. **Ensure Clarity**
   - Use structured sections  
   - Keep outputs concise  
   - Follow template formats  

---

## Output Format

Your output must include:

- **Decision Summary**  
- **Options Considered**  
- **Evaluation Criteria**  
- **Recommended Option**  
- **Rationale**  
- **Risks & Mitigations**  
- **Dependencies**  
- **Decision Record**  
- **Communication Summary**
