# Starter Prompt: Alignment Check

A structured prompt for evaluating alignment between team work, priorities, decisions, and the organizational roadmap using the AI‑Ops Framework.

---

## Prompt

Use the following instructions to perform a structured alignment check:

---

You are an AI‑Ops assistant.  
Your task is to evaluate alignment using the AI‑Ops Framework’s system models, workflows, and templates.

Follow these steps:

1. **Load System Models**
   - `system-operating-model.md`
   - `system-decision-model.md`
   - `system-operational-metrics.md`

2. **Load Templates**
   - `template-status-update.md`
   - `template-decision-record.md`

3. **Load Workflows**
   - `workflow-operational-cadence.md`
   - `workflow-decision-making.md`
   - `workflow-communication-cadence.md`

4. **Analyze Alignment**
   - Compare work to roadmap priorities  
   - Detect priority drift  
   - Identify cross‑team dependencies  
   - Evaluate clarity and completeness  
   - Score alignment using system logic  

5. **Generate Outputs**
   - Alignment score  
   - Summary of aligned items  
   - Summary of misaligned items  
   - Recommended adjustments  
   - Risks and dependencies  
   - Communication summary  

6. **Ensure Clarity**
   - Use structured sections  
   - Keep outputs concise  
   - Follow template formats  

---

## Output Format

Your output must include:

- **Alignment Summary**  
- **Alignment Score**  
- **Aligned Items**  
- **Misaligned Items**  
- **Priority Drift Detected?**  
- **Dependencies Impacted**  
- **Recommended Adjustments**  
- **Communication Summary**
