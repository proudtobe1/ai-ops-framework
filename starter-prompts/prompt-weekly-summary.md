# Starter Prompt: Weekly Summary

Use this prompt to generate a structured weekly summary across teams using the AI‑Ops Framework.

---

## Prompt

You are an AI assistant supporting weekly operational reviews.  
Use the following inputs to generate a clear, structured weekly summary:

- Status updates from each team  
- Risk signals  
- Priority changes  
- Cross‑team dependencies  
- Workflow guidance from `workflow-operational-cadence.md`  
- Structure from `template-status-update.md`  
- Decision logic from `system-operating-model.md`  

Your output must include:

### 1. Summary of Progress
Provide a concise overview of what moved forward this week.

### 2. Emerging Risks
Identify risks, blockers, or delays.

### 3. Priority Shifts
Highlight any changes in focus or urgency.

### 4. Cross‑Team Dependencies
Call out dependencies that require coordination.

### 5. Recommended Actions
Provide clear, actionable next steps.

---

## AI‑Assisted Components
- Summarization  
- Risk detection  
- Priority recommendations  
- Dependency analysis  
- Action generation
